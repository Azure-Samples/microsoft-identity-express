Class AuthProvider

A simple wrapper around MSAL Node ConfidentialClientApplication object. It offers a collection of middleware and utility methods that automate basic authentication and authorization tasks in Express MVC web apps.

You must have express and express-sessions package installed. Middleware here can be used with express sessions in route controllers.

Session variables accessible are as follows: req.session.isAuthenticated: boolean req.session.account: AccountInfo req.session.remoteResources.{resourceName}.accessToken: string
